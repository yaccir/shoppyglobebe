
# ShoppyGlobeBE

Backend service for **ShoppyGlobe** — an eCommerce platform built with Node.js and Express.



## About

**ShoppyGlobeBE** is the backend application for the ShoppyGlobe eCommerce system.  
It provides RESTful APIs for products, users, carts, orders, authentication, and more — enabling a full shopping experience.

## Features

- User registration and authentication
- Secure login with hashed passwords
- CRUD operations for products
- Cart management
- Order processing
- Role-based access (e.g., admin/user)
- Token-based authentication (JWT)

## Tech Stack

- Node.js
- Express.js
- MongoDB / Mongoose
- JWT Authentication
bcrypt



## Getting Started

### Prerequisites

- Node.js (v16.x or later)
- npm / Yarn


### Installation

```bash
git clone https://github.com/yaccir/shoppyglobebe.git
cd shoppyglobebe
npm install
```


PORT=8086
